Introduction

What is Human-Computer Interaction?

Human-Computer Interaction (HCI) is a scientific domain that refers to intaction between humans and the computers and the research activity of this field.

What is INFOX?

INFOX is a singlepage web aplication, a remake after original web application INFOX (infox.ro) whose purpose is to help people to grow their C/C++ skills through solving C/C++ problems created by other users but verified and approved by INFOX admins.

Intended audicente

This document is intended to everyone interested in HCI and steps of developement of INFOX UI, both developers and users.

Notations

Some word shortcuts will be used during this document. These are: HCI = Human-Computer Interaction, UI = User Interface, UX = User eXperience, SPA = SinglePage Application

UI, UX

What is UI?

UI refers to user interface, the medium an user uses to communicate with the app. INFOX UI is user-oriented and it will be discussed in this document.

What is UX?

UX refers to user experience, the way an user feels and the experiences and the pleasure he has while using the app. Good UX is a consequence of a good UI.

Human Factor

Knowing the user

All decisions, pictures, way of doing things, information arrangement were made by having an eye on who the user might be. Types of users will be discussed in HTML5 personas documents.

Interaction idioms

An user can find interaction idioms like: menus, forms, choices lists, informative panels, writing areas.

Visual metaphors

Visual metaphors are present on INFOX. These make users to navigate easily through app by recognizing the image instead of reading the text.

Design decisions

Some of design decisions have been made through respect (as much as it could be) of empirical laws: Fitt's law, Hick's law, Steering law and the web developement principles (e.g. responsiveness)

Fitt's Law

This law refers to how fast a user can move the mouse to a specific point with a specific size. Because of big UI components, users can easily navigate through components so Fitt's law obtain good scores (low mouse movement time).

Hick's Law

This law predict the decision time for a user to make a set of choices. Speaking about chapters the problems are splitted into, an average user, could obtain a low score (high decision time) on Hick's law because of the high number of choices.

Steering Law

This law refers to how long would take to a user to move the mouse through a path that has "borders". Steering law obtain good scores because of "low number of borders". The only place this law might obtain low scores is when user navigate through a specific problem page, which contains multiple tabs for navigation.

Persona

Thanks to easy-to-use interface, almost everyone can use INFOX SPA. The people who cannot use this app, are the people that are not familiar with computers, mice and romanian language (the only available language). More about personas in HTML5 personas documents.

Aspects concerning design

Design

Speaking about design, INFOX has a user-centred interface. All menus and choices an user could make are as simple as they could be. The UI is friendly with big menu buttons, which make app more user-friendly.

3-click-rule

User has to make only 3 clicks to start solving a problem, and the same number of clicks to update their profile and to view their solved problems list.

Chromatics

On the first touch with the app, INFOX consists in shades of green - a color proven to make people feel rested and secure (very important feelings in a problem-solving app), white - a color that create feelings of space, simplicity and comfort and black (for darker shades). Background color can be modified by the user through the user profile/settings page. For this has been chosen shades of yellow, blue (primary colors), green, purple (secondary colors) and gray - a simple background color that is less stressful for eyes.

Design values

INFOX has all three design values: ethical - is not harmful to anyone, purposeful - help users to grow their C/C++ skills and also achieve their goals, elegant - is a simple, complete and elegant design.

Information architecture

INFOX information organization

The organization of information in INFOX is natural, with familiar navigation items (easy to identify by anyone), easy to understand and use lists of components, linear and hierarchical structure of objects list. More on this topic in model, metodologies and design patterns documents.

Conclusion

INFOX is a problem-solving app that is easy to use (high usability), providing all features an user needs on this type of app (high utility), user-friendly and has a very good user experience. The design is very simple and intuitive, and the user can easily navigate through the app.
The design choices were made with the respect of HCI principles, methods and best practices.
